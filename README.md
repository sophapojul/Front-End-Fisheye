## General info
FishEye is a website that allows independent photographers to showcase their best work.

## Technologies
Project is created with:
* Javascript > EcmaScript 6
* Eslint
* Prettier
* Jsdoc
* Sass
* Vite

## Setup
To run this project, install it locally using npm:
```
npm install
npm run serve
```
